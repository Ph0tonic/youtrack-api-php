# Issues
```php 
// Endpoint
$youtrack->projects()
```

##### `all()`
Returns all projects.

##### `one($projectId)`
Returns a specific project based on the project ID.


## Default return fields
```php
[
   'id',
   'name',
   'shortName'
]
```
!> Go to [Fields](fields.md) to learn more about default return fields.
